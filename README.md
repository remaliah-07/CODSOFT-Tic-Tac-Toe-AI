# CODSOFT-Tic-Tac-Toe-AI
This project is an AI-powered Tic-Tac-Toe game developed as part of the CodSoft Artificial Intelligence Internship.

## Features
- Human vs AI gameplay
- Unbeatable AI using the Minimax Algorithm
- Console-based interface
- Demonstrates game theory and search algorithms

## Technology Used
- Python

## How to Run
python tic_tac_toe_ai.py
How to Test in Jupyter
Open a new notebook.
Paste the code into a cell.
Run the cell.
Play by entering numbers 1–9.
